# Perspectives d'Intégration de l'IA dans RAS-SDNCloudSim

Ce dossier explore comment l'Intelligence Artificielle peut transformer la gestion des ressources et du réseau dans le framework RAS-SDNCloudSim.

---

## 📂 Liste des Propositions AI Détaillées

1. [**VM Placement par Reinforcement Learning (DRL)**](./ai_perspectives/01_ai_vm_placement.md)
   - *Technique* : PPO / DQN.
   - *Objectif* : Remplacer le PSO par un agent apprenant pour le placement optimal.

2. [**Équilibrage Prédictif par LSTM**](./ai_perspectives/02_predictive_load_balancing.md)
   - *Technique* : Réseaux récurrents (Long Short-Term Memory).
   - *Objectif* : Anticiper les congestions et déclencher des migrations proactives.

3. [**Routage Intelligent par GNN**](./ai_perspectives/03_gnn_routing.md)
   - *Technique* : Graph Neural Networks.
   - *Objectif* : Optimiser le routage SDN en considérant la topologie comme un graphe complexe.

4. [**Détection d'Anomalies par Auto-encodeurs**](./ai_perspectives/04_ai_anomaly_detection.md)
   - *Technique* : Apprentissage non-supervisé.
   - *Objectif* : Sécurité automatisée et détection proactive de pannes.
